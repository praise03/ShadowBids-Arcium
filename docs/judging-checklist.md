# Judging Checklist

- Technical depth: Anchor state machine, PDA model, fixed-size confidential compute circuit.
- Arcium relevance: private bid comparison is central to the product, not an add-on.
- Demo quality: full lifecycle from creation to sealed bid to final result.
- UX polish: auction desk, status badges, lifecycle controls, portfolio, leaderboard.
- Security posture: no plaintext bids on-chain, callback authority guard, no double finalization.
- Founder trajectory: sealed auctions can expand into launch allocations, OTC sales, grants, and service marketplaces.
